* Asterisk notes

** Ideas

- Use a tailscale docker

** Using


Build and run via the [[file:justfile][justfile]]:

#+begin_src bash
  just            # list recipes
  just build      # build the docker image
  just run        # run with ./etc-asterisk bind-mounted into the container
  just push       # push to the registry
  just all        # build + push
#+end_src

Alternatively use docker compose (adds persistent volumes for
=/var/lib/asterisk= and =/var/log/asterisk=):

#+begin_src bash
  docker compose up --build
#+end_src

Useful commands inside the Asterisk CLI:

#+begin_src bash
  # Turn on pjsip logger
  pjsip set logger on
  # Show registrations
  pjsip show registrations
  # RTP show settings
  rtp show settings
  pjsip show aors
  pjsip show contacts
  core show channels

  # list avialable codecs
  core show codecs
#+end_src
